# 基于Python的前后端分离二手房交易大数据分析系统

## 技术栈

- **前端**
  - vue3 + Element-UI + Element-Plus + Axios + Router
- **后端**
  - python + FastAPI + SQLAlchemy
- **数据库**
  - MYSQL

# 数据库设计

## config表

```sql
CREATE TABLE `config` (
  `id` bigint(20) NOT NULL AUTO_INCREMENT COMMENT '主键',
  `name` varchar(100) NOT NULL COMMENT '配置参数名称',
  `value` varchar(100) DEFAULT NULL COMMENT '配置参数值',
  `url` varchar(500) DEFAULT NULL COMMENT 'url',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB AUTO_INCREMENT=4 DEFAULT CHARSET=utf8 COMMENT='配置文件';
```

## 管理员表

```sql
CREATE TABLE `admin` (
  `id` bigint(20) NOT NULL AUTO_INCREMENT COMMENT '主键',
  `username` varchar(100) NOT NULL COMMENT '用户名',
  `password` varchar(100) NOT NULL COMMENT '密码',
  `image` varchar(200) DEFAULT NULL COMMENT '头像',
  `role` varchar(100) DEFAULT '管理员' COMMENT '角色',
  `addtime` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '新增时间',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB AUTO_INCREMENT=2 DEFAULT CHARSET=utf8 COMMENT='用户表';
```

## 前端页面

## 管理员端

### 管理员登录页 /admin/login

管理员的登录页面

### 管理员主页 /admin/home

展示管理系统开屏页面

### 用户管理 /admin/user

实现对用户账号的管理

### 个人信息管理 /admin/account

修改管理员的账号,头像,昵称,密码等

### 系统管理 /admin/sysconfig

系统配置更改,修改系统简介,修改大屏轮播图等

### 房价管理 /admin/city

以表格形式展示每条房屋信息,支持单条数据的增删改查

### 资讯管理 /admin/news

以表格形式展示每条咨询条目,支持单条数据的增删改查

### 咨询类别管理 /admin/newstype

管理咨询类型,可以自行修改

### 大数据展板 /admin/board

使用echarts库展示系统的大数据分析结果
